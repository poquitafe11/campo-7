# **App Name**: Brujos

## Core Features:

- Main Menu: Display main navigation screen with areas: Production, Health, Irrigation, Quality Control, Biological Control, Queries, and Summary.
- Production Tracking: Production data entry for tracking harvest yields and planting schedules.
- Health Management: Record data related to plant health, diseases, and pest management activities.
- Irrigation Monitoring: Log irrigation schedules and water usage amounts to manage water resources efficiently.
- Quality Control: Input parameters for tracking the quality of products harvested in the field.
- Biological Control: Track biological agents for managing the crop and controlling pests.
- AI Query Assistant: AI powered tool to answer questions from the user, based on summary data from the other features of the app. The questions will be specific to field data, not general knowledge.

## Style Guidelines:

- Primary color: Deep green (#38A3A5) to reflect growth and nature.
- Background color: Light green (#E0F4F1), subtly desaturated, providing a calm backdrop.
- Accent color: Golden yellow (#FFD700), an analogous color, will draw the user's eye without overwhelming.
- Font: 'Inter', a sans-serif, providing a modern and neutral feel for both headlines and body text.
- Simple, outlined icons representing each management area, as illustrated in the sample image.
- Responsive grid layout to adapt to both portrait and landscape orientations on mobile devices.
